# todos

这个项目主要用于yo的使用和教学。

将会建立多个分支阐述创建一个todo list的过程。

第一步：

### 新建一个空项目
```
mkdir todos
slush yo
cd client
spm install
npm install
cd server
npm install
```
在mac或者ubuntu上需要用`sudo`。

运行空项目：
```
redis-server -port 7777

node app
```
